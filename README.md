# dungeon-rs

A **turn-based dungeon crawler** written in Rust.

---

## Run the bevy frontend

```bash
cargo run --release --bin frontend
```

### Controls

- **WASD**: Move around the dungeon

## Run the TUI

```bash
cargo run --release --bin cli -- play
```

### Controls

- **Q**: Quit the game
- **WASD**: Move around the dungeon
- **Shift** + **WASD**: Attack enemy
- **Space**: Skip turn
